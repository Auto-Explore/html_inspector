FAQ & Documentation

How do I create a new project?

Click the "+ New Project" button in the top right corner. Fill in the project name, description, and team members. Then click "Create" to set up your new project.

How do I assign tasks to team members?

Navigate to the project's task board. Click on a task card, then use the "Assignee" dropdown to select a team member. You can also drag and drop tasks between columns.

How do I export reports?

Go to the Reports section from the main navigation. Select the report type and date range, then click "Export" to download as CSV or PDF.

How do I manage integrations?

Visit Settings > Integrations. You can connect Slack, GitHub, Jira, and other tools. Each integration has its own configuration panel.

How do I set up notifications?

Go to Settings > Notifications. Choose which events trigger notifications and select your preferred channels (email, in-app, Slack).
